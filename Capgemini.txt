O projeto Alcance de Anuncios é desenvolvido em Java
e contém sete classes:
-ModuloConexao.java
-TelaCliente.java
-TelaAnuncio.java
-TelaCalculadora.java
-TelaRelatorio.java
-TelaSobreDialog.java
-TelaPrincipal.java (classe main)
Foi usada a ferramenta Workbench projeto para criar um banco de dados (dbanuncios.db) MySQL com as tabelas tbcriterios, tbanuncios e tbclientes. O diagrama relacional é mostrado no arquivo diagrama.pdf. O arquivo capgeminiProj.sql contém todos os comandos sql para a criação do banco de dados e tabelas. Há uma relação de um para muitos entre as tabelas tbclientes e tbanuncios.
Esse projeto foi compilado com sucesso na IDE Netbeans 12.3,
a qual está instalada na plataforma do Windows 10.
